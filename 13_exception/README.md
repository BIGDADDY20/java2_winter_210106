## 예외처리(Exception)
- 예상치 못한 상황에 대한 처리를 하는 것

## 자주 발생하는 예외
|종류|설명|
|---|---|
|NullPointerException|null이 들어있는 객체에서 값을 참조하려고 시도하는 경우|
|ArrayIndexOutOfBoundsException|배열의 크기를 넘어선 곳을 참조하려고 했을 경우|
|ClassCastException|형변환을 할 수 없는 경우|
